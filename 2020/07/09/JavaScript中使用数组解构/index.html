6. Rest 和 Spread 运算符

我们经常会混淆 Rest 和 Spread 运算符，因为它们都使用相同的 ... 语法。Rest 和 Spread 运算符可以在 Array Destructuring 语句中一起使用:

  • Rest 操作符: 在语句的 左边 使用，用于从数组获取剩余属性
  • Spread 操作符: 在语句的 右侧 使用，用于将属性复制到数组中
