手动构建 Docker 镜像方式

下载镜像

下载官方 CentOS 镜像 官方 CentOS 镜像 hub.docker，
找到 Tag，安装指定版本，复制安装镜像命令，例如 $ docker pull centos:7
执行完毕，使用以下命令查看本地所有的镜像：
$ docker images
